Git add

Agrega cambios en archivos al área de preparación (staging) para que se incluyan en el próximo commit.

Git commit

Registra los cambios en el repositorio local con un mensaje descriptivo.

Git status

Muestra el estado actual del repositorio, incluyendo archivos modificados, añadidos o eliminados, y si hay cambios en el área de preparación.

Git push

Envía los commits del repositorio local al repositorio remoto (por ejemplo, en GitHub).
